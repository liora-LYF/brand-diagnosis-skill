# 品牌诊断 Skill

## 1. Skill 定位

该 Skill 用于完成品牌全案中的「品牌诊断层」，帮助使用者从行业、竞品、用户和品牌自身四个维度，找出品牌当前真正的增长瓶颈，并明确下一阶段最值得优先解决的战略命题。

该 Skill 不是简单的信息整理工具，而是一个咨询公司风格的品牌诊断报告生成器。它强调问题导向、结论先行、事实支撑、分析推导和战略启示，适合在品牌战略、传播策略、内容策略、渠道策略和 Campaign 策略制定前使用。

## 2. 适合使用人群

包括但不限于：

- 品牌营销负责人
- 高级品牌经理
- 品牌策略顾问
- 增长营销负责人
- 内容营销负责人
- IP 商业化负责人
- 新业务品牌负责人
- 年度品牌规划负责人

## 3. 适合使用场景

包括但不限于：

- 品牌全案前期诊断
- 年度品牌规划前
- 品牌升级前
- 新业务品牌定位前
- Campaign 策略制定前
- 内容矩阵规划前
- 渠道策略规划前
- 用户增长策略制定前
- IP 商业化规划前
- 品牌现状复盘
- 竞品压力较大时
- 品牌增长乏力时

## 4. Skill 核心逻辑

该 Skill 按照以下逻辑运行：

```text
先定义问题
↓
看外部环境
↓
看竞争对手
↓
看目标用户
↓
看自身品牌
↓
找出真正问题
↓
判断机会方向
↓
形成战略启示
```

## 5. 最终输出

该 Skill 最终输出一份完整的品牌诊断报告，包含：

1. 诊断目标与核心假设
2. 行业环境分析
3. 竞争格局分析
4. 用户洞察分析
5. 品牌现状诊断
6. 核心问题提炼
7. 机会点判断
8. 战略启示与下一步建议

## 6. PDF 导出说明

报告可以通过 Markdown / HTML / PDF 工作流导出为 PDF。

PDF 样式要求：

- 字体：优先使用 PingFang SC / 苹方体
- 版式：适合咨询公司风格报告
- 页面：A4
- 风格：简洁、专业、清晰、结构化
- 标题层级清晰
- 表格适合阅读
- 重点结论突出

推荐使用 `styles/pdf-style.css` 作为 HTML / PDF 导出的基础样式，并参考 `scripts/export-pdf.md` 中的导出说明。
